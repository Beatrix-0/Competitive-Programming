## Boolean DFS

Boolean DFS is used when you need to find if a path/solution exists or find one valid solution rather than all solutions.

## Problem :

Given a graph, is there a path from node S to node T.

```cpp
#include <bits/stdc++.h>
using namespace std;

#ifndef ONLINE_JUDGE
#include "trace.cpp"
#else
#define dbg(...)
#endif

const int N = 100005;
vector<int> g[N];
bool vis[N];

bool dfs(int u, int target) {
    if (u == target) {
        return true;
    }
    vis[u] = true;
    for (int v : g[u]) {
        if (!vis[v]) {
            if (dfs(v, target)) {
                return true;
            }
        }
    }
    return false;
}

void solve() {
    int n, m;
    cin >> n >> m;
    for ( int i = 0 ; i < m ; i++ ) {
        int u, v;
        cin >> u >> v;
        g[u].push_back(v);
        g[v].push_back(u);
    }

    int S, T;
    cin >> S >> T;

    if (dfs(S, T)){
        cout << "Path exists" << '\n';
    }
    else{
        cout << "No path" << '\n';
    }
}

int32_t main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    int tests = 1;
    // cin >> tests;
    for ( int tc = 1 ; tc <= tests ; tc++ ){
        solve();
    }
    return 0;
}
```

## Note :

DFS is implementing logical OR:

Boolean DFS rule:

- false = “this path failed, keep trying”
- true = “solution found, stop everything”

Never return false inside the loop.
Return false only after ALL children fail.
